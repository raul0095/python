# Python roadmap - rnz

Beginner-Friendly Mini Projects

These build fundamentals without overwhelming you.

1. Number Guessing Game

Concepts: input/output, loops, conditionals, random numbers

Why it’s good: short, but teaches program flow

Add-ons: limit attempts, difficulty modes, high-score saving

2. Simple Calculator (CLI)

Concepts: functions, error handling

Add-ons: add history, support parentheses, turn it into a small REPL

3. To-Do List (Text-Based)

Concepts: lists, file reading/writing

Add-ons: save tasks to a file, add timestamps, create categories

4. Password Generator

Concepts: string manipulation, randomness, functions

Add-ons: strength settings, copy-to-clipboard, avoid confusing characters

5. Rock-Paper-Scissors vs Computer

Concepts: logic, randomness, simple AI

Add-ons: score tracking, extended rules (lizard/spock)

🧱 Projects That Start Building Real Skills

Once you can complete the first group confidently, go here.

6. Expense Tracker (File or CSV Based)

Concepts: CSV handling, formatting, real-world logic

Add-ons: monthly summaries, simple data visualization

7. Weather CLI App (Using an API)

Concepts: HTTP requests (requests library), JSON

Add-ons: colorized output, caching past results

8. Flashcard App

Concepts: dictionaries, file I/O, scheduling

Add-ons: spaced repetition logic, export/import decks

9. Basic Web Scraper

Concepts: requests, BeautifulSoup, parsing HTML

Add-ons: scrape multiple pages, save results to CSV

10. Guess-the-Word / Hangman

Concepts: loops, lists, checking conditions, basic game design

Add-ons: word difficulty, ASCII art, hints
